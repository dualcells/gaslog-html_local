# Gas Log

A fuel consumption tracker built with HTML, CSS, and JavaScript.

## Features

- Track multiple vehicles with make, model, year, nickname, and colour
- Record fuel receipts with volume, price, date, odometer, and gas station
- Automatic calculation of fuel consumption (L/100km)
- Dashboard with statistics and averages
- Export/import data as JSON
- Optional password encryption for exports
- Works offline - no server required
- Dark/light theme support

## Usage

Simply open `index.html` in a web browser. No installation or server required.

## Data Storage

All data is stored locally in your browser's localStorage. Nothing is sent to any server. Use the Export button to create regular backups of your data.
